# My Todo App

### Getting Started
#### Clone down this repository. You will need node and npm installed globally on your machine.

Installation:
```bash
  npm install 
```
To Start Server:
```bash
  npm run dev 
```
To Visit App:
```bash
  localhost:3000
```

Structure
```
App
└── src
    ├── assets
    │   ├── styles
    │   ├── images
    ├── components
    │   ├── ui component
    ├── config        
    │   ├── query key           (react query key)
    ├── hooks                   (custom hooks)
    ├── layouts                 (layout components)
    ├── services 
    │   ├── api service
    └── utils
```
